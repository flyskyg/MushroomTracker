# Pikmin Bloom Mushroom Tracker 🍄

一個專為《Pikmin Bloom》玩家設計的簡約、高效蘑菇重生計時器。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🌟 核心功能

- **自動計算重生時間**：只需輸入戰鬥剩餘時間，系統會自動加上 **4 分鐘 30 秒** 的延遲，精確計算出下一個蘑菇重生的時刻，並直接在卡片上標示出重生的「實際時間」。
- **全新地點按鈕網格**：
  - **3 欄式設計**：更直覺的網格佈局，支援大字體顯示。
  - **即時人數預覽**：地點按鈕上會直接顯示目前已設定的人數，一目瞭然。
- **人數追蹤功能**：
  - **一鍵設定**：新增 `5` / `10` / `15` / `20` 人快速設定按鈕。
  - **精確調整**：配有 `+` 與 `-` 按鈕，方便隨時微調。
- **分享連結**：點擊「分享連結」按鈕產生 URL，隊友開啟後可看到相同的倒數計時（以絕對時間戳計算，時間完全同步）。
- **左右佈局自由切換**：新增一鍵切換功能，可依據習慣將「輸入區」與「倒數區」進行左右互換。
- **動態倒數介面**：
  - 🔵 **淺藍高光**：點擊左側地點準備輸入時間或人數時，右側對應的卡片會自動亮起淺藍色背景與外框。
  - 🟢 **綠色**：表示蘑菇已重生（顯示已重生時間）。
  - 🔴 **紅色脈動**：最後一分鐘倒數，提醒您做好準備。
- **一鍵徹底清除**：點擊清除所有通知時，會連同人數與所有狀態徹底歸零，不再殘留綠框。
- **柔和音效通知**：使用 Web Audio API 生成柔和的合成音效，在重生瞬間提醒您。
- **資料自動保存**：即使重新整理頁面，計時進度與人數記錄也會完整保留。
- **每月任務參考**：內建每月任務清單，點擊查看當月目標。

## 🚀 如何使用

1. **啟動音效**：首次進入頁面，請點擊畫面任意處以解鎖音效。
2. **選擇地點**：點擊左側網格中的地點按鈕。
3. **設定人數與時間**：
   - 點擊 `5/10/15/20` 快速設定人數，或使用 `+/-` 調整（按鈕在左側，點後自動跳回時間輸入框）。
   - 輸入剩餘時間（格式：`HHMMSS`，如 `5930` = 59分30秒）。
   - 按下「設定倒數計時」或 Enter 鍵。
4. **追蹤進度**：右側會即時顯示所有倒數卡片，並可透過上方按鈕切換左右佈局。
5. **分享給隊友**：點擊右上角「分享連結」按鈕，複製 URL 傳給隊友，對方開啟後即可同步倒數。

## 🛠 技術棧

- **HTML5 / CSS3 / JavaScript (ES6+)**
- **Bootstrap 5**：響應式佈局。
- **Web Audio API**：用於動態生成無損提示音。
- **Google Fonts**：Inter & JetBrains Mono 提供極佳的閱讀體驗。

## 📦 安裝與運行

不需要安裝任何依賴。

1. 克隆或下載此倉庫。
2. 直接在瀏覽器中打開 `MushroomTracker.html` 即可開始使用。
