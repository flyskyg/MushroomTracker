# Pikmin Bloom Mushroom Tracker 🍄

一個專為《Pikmin Bloom》玩家設計的簡約、高效蘑菇重生計時器。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🌟 核心功能

- **自動計算重生時間**：只需輸入戰鬥剩餘時間，系統會自動加上 **4 分鐘 30 秒** 的固定延遲，精確計算出下一個蘑菇重生的時刻。
- **人數追蹤功能**：
  - **快速加人數**：點擊**地點名稱**可直接快速增加 5 人。
  - **微調按鈕**：每個地點配有 `+` 與 `-` 按鈕，方便精確調整人數。
- **動態倒數介面**：
  - 🟢 **綠色**：表示蘑菇已重生（顯示已重生時間）。
  - 🔴 **紅色脈動**：最後一分鐘倒數，提醒您做好準備。
  - ⚪️ **簡約卡片**：清晰展示各個地點的倒數狀態與人數。
- **柔和音效通知**：使用 Web Audio API 生成柔和的合成音效，在重生瞬間提醒您，無需一直盯著螢幕。
- **資料自動保存**：使用 `localStorage` 技術，即使重新整理頁面或關閉瀏覽器，您的計時進度與人數記錄也不會遺失。
- **每月任務參考**：內建每月任務清單（如：四月花朵任務等），方便點擊查看。
- **離線可用**：純前端單頁應用（SPA），可直接下載 HTML 檔案在瀏覽器打開使用。

## 🚀 如何使用

1. **啟動音效**：首次進入頁面時，請點擊畫面任意處以解鎖瀏覽器的音效限制（或是點擊「測試音效」）。
2. **輸入人數**：
   - 點擊「地名」增加 5 人，或使用 `+/-` 按鈕調整。
3. **輸入時間**：
   - 在左側清單中找到對應的地點。
   - 輸入目前顯示的**戰鬥剩餘時間**（格式：`HHMMSS`，例如 `5930` 代表 59 分 30 秒）。
   - 按下「設定」或 Enter 鍵。
4. **自動加時**：系統會自動計算 `剩餘時間 + 4:30` 並開始倒數。
5. **追蹤進度**：右側會即時顯示所有正在倒數的蘑菇。

## 🛠 技術棧

- **HTML5 / CSS3 / JavaScript (ES6+)**
- **Bootstrap 5**：響應式佈局。
- **Web Audio API**：用於動態生成無損提示音。
- **Google Fonts**：Inter & JetBrains Mono 提供極佳的閱讀體驗。

## 📦 安裝與運行

不需要安裝任何依賴。

1. 克隆或下載此倉庫。
2. 直接在瀏覽器中打開 `MushroomTracker.html` 即可開始使用。
